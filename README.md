# 企业专利全生命周期管理系统 - 演示版

纯前端演示，无需服务器，双击 `demo.html` 即可在浏览器打开，输入任意密码即可进入。

---

## 项目概述

企业专利全生命周期管理系统，是一套面向企业专利管理人员的一站式工具，覆盖专利从**撰写 → 申请 → 审查 → 授权 → 维护 → 终止**的全流程管理。核心解决专利年费逾期、状态跟踪混乱、数据分散等痛点。

---

## 核心亮点

- **全生命周期跟踪** — 12 种权利状态流转（撰写中→已申请→形式审查中→……→专利权生效→已终止），每个状态配有对应的待办事项和附件要求
- **自动费用计算** — 根据专利类型、费减比例、年度自动计算应缴年费、申请费、授权登记费等，逾期自动预警
- **状态流转指引** — 每个状态到下一状态的操作步骤 + 所需附件 + 费用一目了然，用户按指引操作即可
- **仪表盘可视化** — ECharts 展示资产构成饼图 + 动态存量柱状图 + 费用预算查询
- **Excel 批量导入/导出** — 支持 .xlsx 模板导入，自动去重校验，也支持按字段导出
- **数据安全** — 应用密码锁 + 密保问题找回 + 多次失败冷却机制
- **数据备份与恢复** — 一键备份 + 保留最近 5 次快照 + 回收站 30 天自动清理
- **支持两种运行模式** — 可打包为桌面 EXE（Electron），也可作为 Web 服务部署

## 项目结构

```
项目根目录/
├── server.js              ← Web 服务端入口（Node.js HTTP）
├── main.js                ← Electron 主进程入口
├── preload.js             ← Electron 预加载脚本（IPC 桥接）
├── package.json           ← 项目配置与依赖
│
├── src/
│   ├── index.html         ← 应用入口页面
│   ├── css/
│   │   └── style.css      ← 全局样式
│   └── js/
│       ├── api.js          ← API 适配层（Web: fetch / Electron: 跳过）
│       ├── login.js        ← 密码锁、首次设置、忘记密码
│       ├── workbench.js    ← 工作台（专利 CRUD、费用任务、附件）
│       ├── datahub.js      ← 数据中心（Excel 导入/导出、回收站）
│       ├── dashboard.js    ← 仪表盘（ECharts 图表、预算查询）
│       ├── db-migration.js ← 数据迁移（仅 Electron）
│       └── ...             ← 其余前端逻辑
│
├── src/utils/
│   ├── database.js         ← 数据库管理（sql.js SQLite 封装）
│   └── seed_data.js        ← 种子数据（年费标准、状态流转规则）
│
├── data/
│   └── patent.db           ← SQLite 数据库文件（实际数据）
│
├── demo.html               ← 演示版入口（本 demo）
├── demo-mock-api.js        ← 演示版模拟后端
└── demo-lib/
    ├── echarts.min.js      ← 图表库
    └── xlsx.full.min.js    ← Excel 处理库
```

## 技术栈

| 类别 | 技术 | 用途 |
|------|------|------|
| **运行时** | Node.js | 后端运行环境 |
| **Web 服务** | Node.js 内置 `http` 模块 | 静态文件 + REST API |
| **桌面壳** | Electron | 打包为 Windows EXE |
| **数据库** | sql.js (SQLite WebAssembly) | 本地数据库，无需安装 |
| **前端** | 原生 HTML + CSS + JavaScript | 无框架，轻量高效 |
| **图表** | ECharts 5 | 饼图、柱状图 |
| **Excel** | SheetJS (xlsx) | 导入/导出 .xlsx |
| **密码** | Web Crypto API (SHA-256) | 前端密码哈希 |
| **打包** | electron-builder + pkg | EXE 打包 |
